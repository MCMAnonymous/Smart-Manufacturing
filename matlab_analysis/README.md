# Dataset Analysis -- MATLAB Scripts

## Overview

This directory contains MATLAB tools for analyzing the data traffic generated in the industrial plant scenario. The provided scripts allow message identification, filtering, temporal analysis, and visualization of production-related communication patterns.

------------------------------------------------------------------------

# analyze_dataset.m

The `analyze_dataset.m` file is a MATLAB script that analyzes the data traffic generated in the considered industrial scenario. The script includes detailed comments to facilitate understanding of each part of the code.

Using this script, you can:

### 1. Identify the Messages Present in the Dataset

The script identifies the messages described in Table III of the referenced article that appear in the dataset:

-   ACK
-   AGVStatusReport
-   CameraData
-   ConveyorState
-   CraneStatusReport
-   FeedCommand
-   FeedPressCommand
-   FeedPressRequest
-   FeedRequest
-   MaterialRequest
-   PickMaterial
-   PickMaterialRequest
-   PressState
-   PressStatistics
-   ProcessFinalized
-   QualityControlStatistics
-   QualityResult
-   RemoveProduct
-   RobotState
-   RobotStatistics
-   SensorState
-   ServeMaterialCommand
-   SheetInQualityControlArea
-   StartImageCapture
-   StorageStatistics
-   TransitMaterial
-   TransportCommand
-   TransportCompleted
-   TransportStatistics

The script also identifies the components acting as the origin and destination of the generated data.

### 2. Filter Data by Press Line

The script allows filtering of the data generated by the components belonging to each press line.

Components are identified by the suffix `#1`, `#2`, or `#3` appended to their names. For example:

-   RobotController 1 #1
-   FeedLineRobot 2 #3

### 3. Filter Periodic and Aperiodic Messages

The script enables separation of:

-   Periodically generated data
-   Event-driven (aperiodic) data

### 4. Analyze Temporal Generation Patterns

The script can be used to search for temporal generation patterns in the data messages produced by components within the press lines.

### 5. Represent the Data Produced in a Press Line during the Manufacturing of a Steel Sheet

The script generates a figure representing the data generation timeline associated with the production of a single steel sheet (corresponding to Figure 9.b in the referenced article).

### 6. Represent the Total Size and Generation Time of the Data Produced in a Single Press Line

The script can also generate a figure representing the generation time of all data produced by the components within a press line.

# represent_data_time.m 

function [t_uniq, data_agg]=represent_data_time(time, data, time_ini, time_end, S)

The `represent_data_time` function visualizes:

-   The total data size (data_agg)
-   The generation time (t_uniq)

within a specified time interval \[time_ini, time_end\].

data_agg and t_uniq are obtained from arrays data and time that contain the size of the data and generation time of messages generated. 

The figure additionally includes the time instants at which robotic arm controllers R1 or R2 transition to the BusyHandling state. These state transitions are extracted from S, a table containing the data read from data_states.csv.

## Example Usage

``` matlab
represent_data_time(T_line1_aperiodic.Timestamp, ...
                    T_line1_aperiodic.DataSize, ...
                    18000, ...
                    18500, ...
                    S);
```

T_line1_aperiodic is obtained after executing the `analyze_dataset` script.
